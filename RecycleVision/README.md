# ♻️ RecycleVision - Garbage Image Classification

RecycleVision is a Deep Learning based web application built using Streamlit.  
It classifies garbage images into different categories such as:

- Cardboard
- Glass
- Metal
- Paper
- Plastic
- Trash

## Features
- Image upload functionality
- Real-time prediction
- CNN-based deep learning model
- Data visualization dashboard
- Clean UI using Streamlit

## Technologies Used
- Python
- TensorFlow / Keras
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Model Used
The final model selected was MobileNet due to higher accuracy compared to EfficientNet.
