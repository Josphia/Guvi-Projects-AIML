📚 Audible Insights — Book Recommendation System

This project builds an intelligent book recommendation system using machine learning and NLP. It analyzes book details like descriptions, genres, ratings, and reviews to suggest personalized book recommendations.

Users can explore popular books, top authors, genre trends, and get content-based recommendations through an interactive Streamlit app.

🚀 Features

    🔹 Content-based book recommendations

    🔹 Genre-based browsing

    🔹 Popular books & top authors analysis

    🔹 Ratings and listening time insights

    🔹 Interactive dashboard with visualizations

🛠 Tech Stack

Python • Pandas • Scikit-learn • NLP • Streamlit
